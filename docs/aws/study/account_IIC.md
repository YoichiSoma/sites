# AWS IAM Identity Center について
[前項]ではAWS OUを利用することにより、請求先の一本化やAWSアカウントの一元管理ができることを説明した。   
ただし、実際にユーザが利用するためには更にIAMアカウント管理が必要となる。  
そこで"AWS IAM Identity Center(旧AWS SSO)"を利用することにより、まとめて管理できるのでそちらの説明を行う。
